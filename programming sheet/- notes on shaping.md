##### \- notes on shaping :

سؤال ممتاز جداً! فهم "القواعد العامة" هو السر الذي يجعلك قادراً على رسم أي شكل مهما كان معقداً دون الحاجة لحفظ الكود. 



جميع مسائل طباعة الأشكال (النجوم أو الأرقام) تعتمد على ما يسمى بـ \*\*"الحلقات المتداخلة" (Nested Loops)\*\*. 



إليك \*\*الـ 5 قواعد الذهبية\*\* لحل أي مسألة أشكال:



\### القاعدة 1: الهيكل الأساسي ثابت دائماً

أي شكل ستقوم برسمه يتكون من هذا القالب الأساسي:

```cpp

for (int i = 1; i <= number\_of\_rows; i++) {   // 1. الحلقة الخارجية (للصفوف)

&#x20;   

&#x20;   for (int j = 1; j <= ?????; j++) {        // 2. الحلقة الداخلية (للأعمدة/الطباعة)

&#x20;       cout << "\*";                          // 3. أمر الطباعة (بدون سطر جديد)

&#x20;   }

&#x20;   

&#x20;   cout << endl;                             // 4. النزول لسطر جديد بعد نهاية كل صف

}

```



\### القاعدة 2: `i` للصفوف، و `j` للأعمدة

\* \*\*الحلقة الخارجية (`i`):\*\* تحدد عدد الأسطر (الارتفاع). إذا كان الشكل يتكون من 5 أسطر، فإن `i` ستبدأ من 1 وتنتهي عند 5.

\* \*\*الحلقة الداخلية (`j`):\*\* تحدد ما سيتم طباعته في السطر الواحد (العرض). 



\### القاعدة 3: الشرط في حلقة `j` هو الذي يرسم الشكل!

السر كله يكمن في إجابة هذا السؤال: \*\*"كم نجمة أريد أن أطبع في الصف رقم `i`؟"\*\*

\* \*\*المربع / المستطيل:\*\* عدد النجوم ثابت في كل صف ولا يعتمد على `i`.

&#x20; \* \*الشرط:\* `j <= N` (حيث N هو طول الضلع).

\* \*\*المثلث القائم العادي (يزيد):\*\* في الصف الأول نجمة، في الثاني نجمتين.. إذن عدد النجوم يساوي رقم الصف `i`.

&#x20; \* \*الشرط:\* `j <= i`

\* \*\*المثلث المقلوب (ينقص):\*\* في الصف الأول 5 نجوم، في الثاني 4.. إذن النجوم تتناقص.

&#x20; \* \*الشرط:\* `j <= (N - i + 1)` 



\### القاعدة 4: الأشكال المائلة أو الهرمية تحتاج "مسافات"

إذا كان الشكل غير ملاصق لليسار (مثل الهرم)، فهذا يعني أنك تطبع "مسافات وهمية" قبل أن تطبع النجوم.

في هذه الحالة، نحتاج \*\*حلقتين داخليتين\*\* (واحدة للمسافات، وواحدة للنجوم):

```cpp

for (int i = 1; i <= N; i++) {

&#x20;   // 1. حلقة لطباعة المسافات (عادة تتناقص)

&#x20;   for (int j = 1; j <= N - i; j++) cout << " ";

&#x20;   

&#x20;   // 2. حلقة لطباعة النجوم (عادة تتزايد)

&#x20;   for (int j = 1; j <= (2 \* i - 1); j++) cout << "\*";

&#x20;   

&#x20;   cout << endl;

}

```

\*(ملاحظة: معادلة `2 \* i - 1` هي السر في طباعة الأعداد الفردية 1, 3, 5, 7 لبناء الهرم).\*



\### القاعدة 5: ابدأ بالورقة والقلم (تتبع العلاقة الرياضية)

إذا واجهت شكلاً غريباً، ارسم جدولاً صغيراً يربط بين رقم الصف `i` وعدد النجوم (أو المسافات) المطلوبة فيه.

\*\*مثال لشكل الهرم (بافتراض الارتفاع 3):\*\*



| رقم الصف (`i`) | عدد المسافات المطلوبة (`3 - i`) | عدد النجوم المطلوبة (`2i - 1`) |

| :--- | :--- | :--- |

| \*\*1\*\* | 2 | 1 |

| \*\*2\*\* | 1 | 3 |

| \*\*3\*\* | 0 | 5 |



بمجرد أن تستنتج العلاقة الرياضية من الجدول، تضعها مباشرة في شرط الحلقة `j`، وسيعمل الكود بشكل سحري!
